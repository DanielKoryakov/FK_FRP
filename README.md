# FRP Client FK



## И так начнем))

Для установки этого клиента вам необходимо пройте все этапы попорядку!

## Установка

* Скопируйте ссылку расположенную в окне ниже

```
https://gitlab.lgsf.tech/haos/frp-client-fk

```
* Откройте Меню-Настройки-Приложения и в правом нижнем углу нажмите Установить приложение
* Далее в правом вехнем углу нажмите на три точки ... и в открывшемся окне нажмите Репозитории
* Вставьте скопированную ссылку и нажмите Добавить, а затем Закрыть
* Верху в строке Поиск введите FRP Client FK
* Установите приложение
* Перед запуском нужно пройти на вкладку Конфигурация и настроить клиента под ваш сервер
